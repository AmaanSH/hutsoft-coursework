    HTML

    HTML (Hyper Text Markup Language) is the standard markup language for Web pages

    Origin of HTML

    First developed by Tim Berners-Lee in 1990, HTML is short for Hypertext Markup Language. HTML is used to create electronic documents (called pages) that are displayed on the World Wide Web.


    HTML Fundamentals

    Here is an example of a basic page structure for a html web page:

    <!DOCTYPE html>
    <html>
    <head>
    <title>Page Title</title>
    </head>
    <body>

    <h1>I am a large heading!</h1>
    <p>I am a paragraph</p>

    </body>
    </html>

    You're probably wondering what that all means right? Well here is a breakdown of each line and what it is responbile for

    • The <!DOCTYPE html> defines what kind of document it is
    • The <html></html> element is the root element of the HTML page
    • The <head></head> tags contains the meta information about the document
    • The <title></title> tags specify the title for the document - displayed in the browser tab
    • The <body></body> tags contain the visible site content
    • The <h1></h1> element defines a large heading
    • The <p></p> element defines a paragraph

    HTML Tags

    HTML tags let the webpage know how to display a certain element, they are surrounded by angle branckets:

    <tagname>content goes here</tagname>

    • HTML tags usually come in pairs. e.g. <p> and </p>
    • The first tag is the starting tag and the second tag is the end tag
    • The ending tag is written the same as the first tag but with a /

    Tip: Some tags like img (adds an image to the website), br (creates a line break) do not need a closing tag

    In the previous example you saw how we used a h1 tag. Did you know there are many different styles to h1 tags?
