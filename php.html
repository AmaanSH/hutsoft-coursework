    PHP

    Hypertext Preprocessor is a general-purpose programming language originally designed for web development.

    What is PHP?

    PHP is a widely-used open source scripting language that is suited for web development and can be embedded into html. This web page uses PHP on the Contact page add a new comment to the database and show comments.

    PHP Example

    Here is a quick example of some PHP code to display some text in a html file:

    echo "Hello World"

    We use echo "" to produce a piece of html that says what you supplied

    PHP Server Side

    PHP is a server side language and can be used to interact with databases in order to retrieve and set information. Here is an example flow of some html code linking with PHP to interact with a database

    Diagram Explanation

    • User clicks submit in the browser window
    • Submit causes HTTP post method to run, sends data from browser to the webserver
    • Webserver passes the request to the PHP script
    • PHP script runs and interacts with the MySQL database
    • Then heads back to the machine

    Here is another example but with a form in HTML:

    <form action="RunMe.php" method="POST>
    <button action="submit>RUN ME!</button>
    </form>

    The following code is ran in the RunMe.php file when the submit button is pressed:

    <?php echo "This is some PHP returning some HTML to your browser using the echo command."; ?>
